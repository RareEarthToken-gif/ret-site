# Rare Earth Token (RET)

Rare Earth Token (RET) is an ERC-20 digital asset inspired by rare earth elements as a conceptual reference.

## Key Characteristics

- Fixed supply: 30,000,000 RET
- Non-redeemable
- No rights to physical assets
- Not asset-backed
- Not intended as a financial instrument

## Token Details

- Network: Ethereum Mainnet
- Standard: ERC-20

## Disclaimer

RET does not represent ownership of any commodities, securities, or financial instruments.  
It does not provide rights to profits, income, or underlying assets.
